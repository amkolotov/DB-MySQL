Описание базы данных top_hotels

База данных top_hotels разработана для хранения данных об отелях по примеру сайта tophotels.ru.

База данных решает задачи хранения информации об отелях по наименоваию, месту расположения, категорям,
рейтингам, хранения медиа файлов, отзывов и рекомендаций, лайков, подписок, новостей относящихся к 
отелям.

База данных включает в себя следующие таблицы:

1. users - хранит данные зарегестрированных пользователей, включая идентификатор пользователя, 
тип пользователя(турист, турагент, отельер, гид, имя, пол, электронную почту, телефон, пароль, 
дату создания и обновления аккаунта.

2. Таблицы countries, regions и cities предназначены для хранения информации о городах, регионах и 
странах расположения отелей соответственно. 

3. Таблицы  star_categories, type_hotels и type_locations хранят информацию о категории звездности, 
типе размещения и типе расположения отелей соответственно.

4. Таблица отелей хранит информацию об отелях, включая наименование, айди категории звездности, 
айди типа размещения, айди типа расположения отеля, описание отеля, айди города, адресс нахождения, 
контактную информацию, дату строительства, дату реконструкции, дату создания и обновления записи.

5. Таблица categories_media предназначена для распределения фотографий и видео к местам в отелях.

6. Таблица media предназначена для хранения данных о медиафайлов, включая информацию о пользователе, 
добавивщем ее, типе медиафайла, наименование отеля, место в отеле, путь хранения, размер, дату 
создания и дату обновления.

7. Таблица reviews предназначена для хранения отзывов и рекомендаций от пользователей об отелях.

8. Таблицы subcription, likes, ratings хранят информацию о подписках пользователей на отели, 
лайках от пользователей и рейтинга отеля от пользователя.

9. Таблица news предназначена для хранения новостей.
